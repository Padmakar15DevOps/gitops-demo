\# GitOps Deployment Platform



\## Project Overview



This project demonstrates GitOps deployment using Argo CD and Kubernetes.



\## Technologies



\- Kubernetes

\- Argo CD

\- GitHub

\- Docker Desktop

\- Git



\## Project Architecture



GitHub Repository

&#x20;       |

&#x20;       v

&#x20;    Argo CD

&#x20;       |

&#x20;       v

&#x20;Kubernetes Cluster

&#x20;       |

&#x20;       v

&#x20;  Nginx Application



\## Features



\- GitOps Deployment

\- Continuous Synchronization

\- Kubernetes Deployment

\- Automatic Scaling

\- Argo CD Dashboard



\## Deployment



1\. Install Docker Desktop

2\. Enable Kubernetes

3\. Install Argo CD

4\. Push Kubernetes manifests to GitHub

5\. Create Argo CD Application

6\. Sync Deployment



\## Verification



\- kubectl get pods

\- kubectl get svc

\- kubectl get deployments

\- Argo CD Dashboard

\- Nginx Browser Page
